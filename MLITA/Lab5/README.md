#++Задача 5.3 (5):  


Найти суммарное число цифр во всех целых числах от M до N включительно.  


Ввод из файла input:  
В единственной строке находятся целые числа M и N (1 ≤ M ≤ N ≤ 10^15 ).  


Вывод в файл output:  
В единственную строку вывести суммарное число цифр.  


Примеры:  


Ввод 1:  
1 2  

Вывод 1:  
2  


Ввод 2:  
2 11  

Вывод 2:  
12
